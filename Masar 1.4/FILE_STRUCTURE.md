## How to Get Started

1. **Read First**: `QUICK_START_AUTH.md` (5 minutes)
2. **Install**: `npm install`
3. **Start Backend**: `npm run server` (one terminal)
4. **Start Frontend**: `npm start` (another terminal)
5. **Test**: Follow `TESTING_INSTRUCTIONS.md`
6. **Reference**: `MONGODB_AUTH_SETUP.md` for details

## Database Schema

### User Collection (MongoDB)
```javascript
{
  _id: ObjectId,
  email: String (unique),
  password: String (hashed),
  name: String,
  phone: String,
  balance: Number (default: 0),
  createdAt: Date
}
```

## API Routes

```
POST   /auth/signup              - Create account
POST   /auth/login               - Authenticate
GET    /auth/user/:userId        - Get profile
PUT    /auth/user/:userId/balance - Update balance
```

## Authentication Flow

```
App Opens
    ↓
Is User Logged In?
    ├─ NO  → Login/Signup Screen
    │        ├─ Sign Up → Create account (balance 0)
    │        └─ Login → Enter app
    │
    └─ YES → Home Screen (with all tabs)
             └─ Settings/Profile → Logout
```
